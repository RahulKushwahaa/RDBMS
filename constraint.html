Constraint :-

Set of rules. Used to maintain data integrity.
  1. Not null constraint
  2. Primary key constraint
  3. Unique key constraint
  4. Foreign key constraint
  5. Cheak constraint

Not null constraint :-

Used to make field mandatory.

Primary key constraint :-

Primary kay column contains unique value. Primary kay column can't contains NULL value.
One table may have only one primary kay. A good table must must contain primary key.
