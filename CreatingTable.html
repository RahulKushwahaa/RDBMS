Creating Table :-

CREATE TABLE - This command is used to create a new table.
Syntax :-

CREATE TABLE tableName(fieldName type[(size)] [,fieldname type[(size)],...] ;

TableName - Any valid identifier
fieldName - Any valid identifier
size - Number
type - Any valid oracle data type. Orcale comman data type:
for numeric value - Number
for string value - char, varchar2, nchar, nvarchar2
Ex :-
Create a table in which we can store the name, gender, address, mobileno of our friends.
CREATE TABLE friends(name varchar2(30), gender char, address varchar2(50), mobileno char(10));
Create a table for storing students records as: name, class, roll, hin, eng, math, science.
CREATE TABLE students(name varchar2(30), class number(2), roll number(3), hin number(3), eng number(3), math number(3), science number(3));

View structure of table :-

DESCRIBE - This command is used to view the structure of pre-created table.
Syntax :-

DESC[RIBE] tableName ;

Ex :-
DESCRIBE students;
or,
DESC students;

Inserting data in table :-

INSERT - This command is used to insert data in pre-created table.
Syntax :-

INSERT INTO tableName VALUES(field1Value, field2Value, field3Value,...) ;

Note :-String value must be enclosed within single quotes('').
Ex :-
INSERT INTO friends VALUES ('rahul kumar','m','gaya',9934320778);
OUTPUT - 1 Row created
INSERT INTO students VALUES ('neha kumari',10,45,56,23,56);

View records of table :-

SELECT - This command is used to view the records of table.
Syntax :-

SELECT * FROM tableName ;

Ex :-
SELECT * FROM friends;
SELECT * FROM students;

Inserting specific fields value in table :-

Syntax :-

INSERT INTO tableName[(fieldName[,fieldName,...])] VALUES(fieldValuesSepratedByComma);

Creating new user in oracle :-

Syntax :-

CREATE USER userName IDENTIFIED BY password;

Ex :-
CREATE USER rahul IDENTIFIED BY rahul12345;
message: new user created
After Creating user grant access to that use as:

GRANT DBA TO userName;

Change password of user :-

Syntax :-

ALTER USER userName IDENTIFIED BY new password;

View records of table :-

Syntax :-

SELECT {columnName[,columnName]|*} FROM tableName;

{ } => Is not part of syntax.
{ } => Represents mandatory valur.
| => or value.
Ex :-
x | y | z => Any one value should be used.

Here ( * ) represents all columns.

Deleting table :-

DROP TABLE - This command is used to delete the pre-created tables. It will delete table with its records.
Syntax :-

DROP TABLE tableName;

Ex :-
DROP TABLE students;

Deleting user ( only in oracle ) :-

ALTER TABLE - This command is used to modify the structure of table.

Add new column to table :-

Syntax :-

ALTER TABLE tableName ADD columnName type [(size)];

or,

ALTER TABLE tableName ADD (columnName type[(size)] [,columnName[(size)]]);

Modifying type of size of columns :-

Syntax :-

ALTER TABLE tableName MODIFY pre-created columnName type(size);

Deleting column of table :-

Syntax :-

ALTER TABLE tableName DROP COLUMN columnName;

Updating record(s) in table :-

UPDATE - This command is use to update record(s) in table.
or,
This command will sets the given value in specified field in rows that satisfies the condition.
Syntax :-

UPDATE tableName SET fieldName = value [WHERE condition];

Ex :-
UPDATE result SET marks=90 WHERE roll=10;

Deleting records from table :-

DELETE -This command is used to delete records from table.
Syntax :-

DELETE FROM tableName [WHERE condition];

Ex :-
DELETE FROM result WHERE roll=10;
It will delete all records having roll value 10.

DELETE FROM result;

It will delete all records of table.

Deleting all records of table :-

TRUNCATE - This command is used to delete all record(s) of table. This command EMPTY the table.
Syntax :-

TRUNCATE TABLE tableName;

All records will be deleted. Table will be empty.

Deleting the table :-

DROP TABLE - This command is used to delete the table.
Syntax :-

DROP TABLE tableName;

Ex :-
DROP TABLE result;

Fetch record(s) of table :-

SELECT - Used to fetch records from table.
Syntax :-

SELECT columnName ToBeFetch FROM tableName;

Ex :-
SELECT srno FROM students;
it will list serial number of all records.
SELECT srno, name, class FROM students;
it will list serialNumber, name and class of all records.
SELECT srno, name, class, roll FROM students;
it will list serialNumber, name, class and roll (i.e. all columns) of all records in table.
SELECT * FROM students;
it will list all columns of all records in table.

Selecting specific record(s) from table (Filtering table) :-

Syntax :-

SELECT columnName TOFetch | * FROM tableName [WHERE condition];

it will list all records for which condition will be true.
Ex :-
SELECT * FROM students WHERE roll < 50;
it will list all records which roll number will be less than 50.

Fetching records in sorted order :-

Syntax :-

SELECT {columnName | *} FROM tableName [WHERE condition] [ORDER BY columnName [ASC | DESC] [,columnName [ASC | DESC],...]];

Like operator :-

Like operator is used with strino match pattern.
Like operator is uses two wildcard character ( _ , % ). _ represent a single character. % represents zero, one or more than one characters.
name like ' _ ' -> it will true for all records having name field one character value.
name like ' _ _ ' -> it will true for records which name field has exact 2 character value.
name like ' a _ ' -> it will for records which name field value beings with ' a ' and must be exact 2 character value.
name like ' a % '-> it will true for records which name field value beigins with symbol ' a '.
